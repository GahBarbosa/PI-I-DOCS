# 1. H - Como?

O sistema será desenvolvido com foco em simplicidade e eficiência, utilizando as seguintes tecnologias:

- **Backend**: Lógica desenvolvida em **React**, responsável pelo processamento de dados e integração com o banco de dados.
- **Banco de dados**: Utilização de um banco relacional com **SQL**, para armazenar dados de produção, estoque e histórico de transações.
- **Frontend / Relatórios**: A interface será construída com **Power BI**, possibilitando visualizações interativas e responsivas, facilitando a manipulação e análise dos dados pelos usuários.
