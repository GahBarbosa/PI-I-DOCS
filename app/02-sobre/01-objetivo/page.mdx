# Objetivos do Projeto

O projeto tem como objetivo geral desenvolver um sistema de apoio à gestão da AMG Confecções, permitindo o acompanhamento do fluxo de pedidos, controle de insumos e fornecedores, e análise de dados operacionais.

## Objetivos Específicos

- Facilitar o **cadastro e acompanhamento de pedidos**;
- Permitir o **gerenciamento de clientes, fornecedores, produtos e insumos**;
- Implementar **indicadores de desempenho**;
- Gerar **relatórios analíticos via Power BI** para apoiar decisões estratégicas;
- Promover a **rastreabilidade dos processos produtivos**.
