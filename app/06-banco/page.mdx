# Banco de Dados

Neste capítulo, apresentamos a modelagem do banco de dados desenvolvida para o projeto, explicando como os dados estão organizados, relacionados e estruturados.

Você irá encontrar os diagramas conceitual, lógico e físico (acompanhado de um dicionário de dados detalhado). Também estão incluídas as regras de negócio relacionadas às entidades, o script de criação do banco e exemplos de consultas SQL, visando tornar este módulo uma referência completa para entendimento do funcionamento do projeto.
