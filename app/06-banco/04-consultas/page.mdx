# 4. Exemplos de Consultas SQL

O objetivo é demonstrar como acessar e manipular as informações armazenadas, utilizando SQL de forma clara e eficiente.

Cada exemplo acompanha uma explicação do que a consulta realiza e em quais situações ela pode ser útil, servindo tanto como guia de uso quanto como base para a construção de novas queries, conforme as necessidades.

## Exemplo 1 - Listar todos os clientes
> Essa consulta retorna os dados principais dos clientes ordenados pela data de cadastro, do mais recente ao mais antigo.

```sql
SELECT id, nome, cpf_cnpj, email, data_cadastro
FROM dim_cliente
ORDER BY data_cadastro DESC;
```

## Exemplo 2 - Buscar produtos com estoque
> Exibe os produtos que possuem unidades em estoque, permitindo visualizar quais itens estão disponíveis para venda.

```sql
SELECT id, nome, estoque, preco_unitario
FROM dim_produto
WHERE estoque > 0
ORDER BY nome;
```
