# Projeto Integrador I

Este projeto foi desenvolvido como parte da disciplina de **Projeto Integrador I**, no **primeiro semestre** do curso de **Big Data para Negócios** da **FATEC Ipiranga**.

O trabalho apresenta o desenvolvimento de um sistema para controlar o fluxo de trabalho de uma empresa do setor têxtil. O sistema permite o gerenciamento de clientes, fornecedores, produtos, insumos e pedidos, além da geração de relatórios integrados com Power BI para análise de desempenho e apoio à tomada de decisão.

O principal objetivo deste projeto é proporcionar **rastreabilidade** nos processos internos da empresa, permitindo que gestores e funcionários tenham uma visão clara e estruturada das operações. Com isso, espera-se facilitar a criação de **estratégias baseadas em dados** reais da operação.

O sistema é voltado para **uso interno**, contribuindo diretamente para a organização, agilidade e controle dos fluxos produtivos.
